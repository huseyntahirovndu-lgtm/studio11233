# **App Name**: İstedad Mərkəzi

## Core Features:

- Student Profile Creation: Students can create detailed profiles showcasing their skills, projects, achievements, and social links.
- Advanced Search and Discovery: Organizations can search for students based on various criteria such as faculty, course, category, skills, and GPA.
- Talent Scoring Tool: The system evaluates student profiles to calculate a composite score representing overall talent, which can be used for ranking and filtering using an AI-powered tool.
- Organization Dashboard: Organizations have a dashboard to view profiles, save favorites, and send messages to students.
- Admin Panel: Administrators can manage students, organizations, categories, and statistics.
- Rating System: Top talent list based on student scores, broken down by faculty and category.

## Style Guidelines:

- Primary color: Soft teal (#40E0D0) for a fresh, modern feel.
- Background color: Light gray (#F0F0F0) for a clean and spacious layout.
- Accent color: Coral (#FF7F50) to highlight key elements.
- Body font: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth or personality.
- Headline font: 'Playfair' with 'PT Sans' as the body font.
- Use 'lucide-react' icons to maintain a consistent style. These icons will improve usability.
- Employs card-based layout. Rounded corners give elements a softer edge.
